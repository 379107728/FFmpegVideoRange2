# FFmpegVideoRange2
 
## 此项目是android 仿照剪映一些功能，暂时只有预览条的精准快速抽帧显示，以及滑动过程中与播放逐帧显示，我观察了下，基本只有大厂的剪辑软件中会有这样的功能
## 并且没有找到开源的项目有相关功能。有些仅仅是非精准抽帧或者只是关键帧，有些速度很慢。

### demo 需要在内部存储根目录创建一个testVideo目录，将测试视频放到这里
